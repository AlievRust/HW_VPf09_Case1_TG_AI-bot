# Telegram AI-бот на Alice AI

Приватный Telegram-бот на Python, который:

- отвечает через Alice AI;
- хранит контекст последних сообщений по каждому пользователю;
- переключает роли через `prompts.json`;
- умеет сбрасывать контекст командой `/reset`;
- умеет генерировать изображения командой `/img`.

## Установка

1. Создайте и активируйте виртуальное окружение.
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Скопируйте `.env.example` в `.env`.
4. Заполните:

- `TELEGRAM_BOT_TOKEN`;
- `YANDEX_CLOUD_FOLDER`;
- `YANDEX_CLOUD_API_KEY`;
- `ALLOWED_USER_IDS`.

5. При необходимости отредактируйте `prompts.json`.

## Запуск

```bash
python -m tg_ai_bot.main
```

## Команды

- `/start` - краткая справка и текущая роль;
- `/roles` - список доступных ролей;
- `/role <ключ>` - переключение роли;
- `/reset` - сброс контекста и возврат роли по умолчанию;
- `/img <промпт>` - генерация изображения.

## Контекст

По умолчанию бот хранит последние 10 сообщений на пользователя. Значение меняется через `MAX_CONTEXT_MESSAGES` в `.env`.

## Файлы состояния

- `prompts.json` - список ролей;
- `.bot_state.json` - локальное состояние диалогов и активных ролей.
