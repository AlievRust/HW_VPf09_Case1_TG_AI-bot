# Проект: Telegram AI-бот на Alice AI

## Назначение

Проект реализует Telegram-бота на Python, который общается с пользователем через модели Alice AI от Yandex Cloud, хранит контекст диалога по пользователю и умеет переключать роли через `prompts.json`.

## Цели

- обеспечить диалоговый режим с памятью последних сообщений;
- дать возможность переключать системные роли без изменения кода;
- добавить отдельную команду генерации изображений;
- хранить рабочие настройки в `.env`.

## Ограничения

- бот работает только для пользователей из allowlist, заданного в окружении;
- состояние диалога хранится локально в JSON;
- реализации без OpenSpec не считаются источником истины для пользовательского поведения.
