Идея: создание тг-бота-агента с подключенной LLM yandex (Alice AI) и поддержкой различных ролевых моделей.
Настройки тг, ллм - в .env файле
Код на Python с использованием aiogram
Поддержка контекста 10 последних сообщений (число настраивается через .env) для каждого пользователя
Разрешенные id пользователей для работы бота в .env: частично env.example уже создан, используй
Роли бота с соответствующими системными промптами в файле prompts.json, например:

Для каждого пользователя бот помнит режим и контекст (используй json), пока пользователь не сбросит
Пользователь после старта может переключать роль бота, а также сбрасывать контекст командой reset
Отдельная ветка - команда img: запрашивает промпт, генерит картинку и возвращает в чат.
Примеры обращений к Alice Art и Alice AI в файле alice.examples - можешь использовать по необходимости
Используй venv и создай requirements.txt для установки зависимостей
